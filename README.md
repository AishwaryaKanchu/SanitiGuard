#Problem
Urban areas face clogged drains, overflowing sewage, and stagnant water after rains, causing health hazards and inconvenience.

#Our Solution
SanitiGuard monitors drains in real-time, predicts issues, and alerts authorities instantly. It uses a live map interface, task tracking, and data-driven insights for smarter city sanitation and also verify tasks.
 
#Features at a Glance
| Feature                    | Benefit                       |
| -------------------------- | ----------------------------- |
| Real-time Drain Monitoring | Detect blockages instantly    |
| Automated Alerts           | Fast municipal action         |
| Interactive Map            | Visualize sanitation hotspots |
| Dashboard                  | Track tasks & updates         |
| Data Insights              | Optimize cleaning schedules   |

#Tech Stack
-Frontend: HTML, CSS, JS
-Backend: Node.js / Python Flask
-Data Handling: CSV
-APIs: Maps API
-Version Control: Git & GitHub
-Google technology: Google Maps JavaScript API


# How to See It Live
1.Clone:
git clone https://github.com/AishwaryaKanchu/SanitiGuard.git
2.Enter folder:
cd SanitiGuard
3.Install & Run:
